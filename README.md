
# mern-stack-blog-app

 mern stack blog application

## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB
  
## Run Locally

Clone the project

```bash
  git clone https://github.com/sam094-dev/blogAppProject
```

Go to the project directory

```bash
  cd blog-app-testbranch
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  node server.js
```
Start the Client

```bash
  //open now terminal
  cd frontend
  npm start
```
